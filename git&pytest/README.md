# Steam Login Automation Test

Проект для изучения автоматизации тестирования с использованием Selenium и Pytest.

## Технологии

- Python 3.13
- Selenium WebDriver
- Pytest
- Faker

## Установка

```bash
pip install selenium pytest webdriver-manager faker
```

## Запуск тестов

```bash
pytest -v -s
```

## Структура проекта

```
git_pytest/
├── git&pytest/
│   ├── test_steam.py      # Тест входа в Steam
│   └── check_locators.py  # Отладочный скрипт
└── README.md
```
